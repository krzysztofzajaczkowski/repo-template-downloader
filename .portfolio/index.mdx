GitHub Action for downloading specified template file and cover image from all user's public repositories. 
Each pair of template/image for repository is stored in path that is combined from parametrized output path and repository name.
This Action is used for building <Underlining highlight><a href="https://github.com/krzysztofzajaczkowski/distributed-portfolio" target="_blank" rel="nofollow noopener noreferrer" aria-label="External Link">distributed portfolio</a></Underlining> project.
